# Data

This folder contains raw output data from the experiments, so the report figures and tables can be reproduced without re-running the notebooks from scratch.

## Files

- `Classical_Shor_Analysis_Results.xlsx` - Classical period-finding results for 2000 semiprimes (4-27 bits), including operation counts, timing, success rates, and RSA extrapolation. Generated by `Comparison analysis circuits/shors algorithm (classical analysis).ipynb`.

## Notes

- Hardware results from IBM Quantum (ibm_fez) are embedded in the `on hardware.ipynb` notebook outputs. Re-running hardware jobs will produce different results due to device noise variation between calibration cycles.
- Simulator results are deterministic given the same Qiskit version and shot count (4096).
